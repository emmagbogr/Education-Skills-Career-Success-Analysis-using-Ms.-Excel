## ğŸ“ˆ Key Insights

* The student population is **perfectly gender-balanced** (200 male, 200 female).
* A total of **1,050 certifications** were obtained, with an average of **3 per individual**.
* **65%** of individuals occupy **mid-level roles**, while only **15%** are in senior positions.
* **Computer Science graduates** dominate senior-level roles and earn the highest starting salaries.
* **Business graduates** dominate overall employment but are underrepresented in senior roles.
* Several fields (Arts, Education, Nursing, Marketing, Psychology, Finance) show **no senior-level representation**, indicating slower career progression.
